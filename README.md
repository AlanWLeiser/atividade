# Atividade margin, padding e border

## Margin
- Margin é uma propriedade CSS que define um espaço entre os elementos ou uma área de margem entre eles.
- Margin é uma abreviação de margin-bottom, margin-top, margin-left, margin-right
### Como usar
- Margin recebe 1, 2, 3 ou 4 valores, eles podem ser pixel, porcentagem ou uma palavra chave
- Exemplo:
`margin: 15px;` 

## Padding
- Padding é o espaço entre o conteúdo do elemento e a borda.
- O comando padding é a abreviação de padding-bottom, padding-top, padding-left, padding-right

### Como usar
- Padding recebe 1, 2, 3 ou 4 valores, eles podem ser pixel, porcentagem ou uma palavra chave
- Exemplo:
`padding: 15px;` 

## Border
- Border define o contorno do elemento.
### Como usar
- O border aceita 3 valores, primeiro a espessura (em pixel), estilo (solid, ...) e por ultimo a cor.
Exemplo:
`border: 1px solid black;`
